Jigsaw Puzzle: File Commands
======

Jigsaw puzzles are my favorite.

In this one, you have 4 file related commands, one in each line. You can either arrange the letters of the command or arrange the picture pieces of the background image.

Either way, complete the puzzle as quick as possible.
